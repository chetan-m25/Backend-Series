* What are API's ?
> Application Program Interface (API) is a set of rules and protocols that enables.
> Different software programs to communicate and exchange data with each other.
> inshort = An API is a bridge between a client and a server.

* What is REST API ?
> A REST API is an API that follows REST rules to send and receive data using HTTP methods.
> A REST API lets clients communicate with servers using standard web methods like GET, POST, PUT, DELETE.
