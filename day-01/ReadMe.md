* What are Packages ? 
> A package is a collection of related classes, functions, or files kept in one place.

* How install Packages ?
> To install packages in js use NPM and in your terminal write command .
  Ex: npm install PackageName
        npm install cat-me
  Packages Website = npmjs.com

* How to use Packages ?
> By using require("PackageName") it is older way.
  require("cat-me")
  import is new one came with ES6.

* node_modules => Contains all installed npm packages.
             => Generated automatically using npm install.

* package-lock.json => Locks exact versions of installed packages.
                    => Ensures the same setup on all systems.

* package.json => Main project configuration file.
               => Lists of project info, dependencies, versions and scripts.

To Run the Code = node FileName.js 
Example: node app.js
